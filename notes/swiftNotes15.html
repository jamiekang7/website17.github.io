Enumerations

Back to Notes

To define an enumeration, use the keyword "enum"

The "enum" defines the type, while the "case" options define the available values allowed with the type.


As seen above, the cases can be defined in two different ways

Once the enumeration has been defined, it can be used like any other type in Swift. Just specify the enumeration type along with the value.

You can use it to define an object even in two different ways.



Control Flow

You can use control flow logic when working with different cases of enumeration.
