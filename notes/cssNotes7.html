text-align

  • text-align is a css property
  • Specifies the horizontal alignment of text

text-decoration

  • text-decoration is a css property
  • Adds an underline, over-line, line-through, to selected text

font-weight

  • font-weight is a css property
  • Bold or light text

font-family

  • font-family is css property
  • Specifies the font for an element

<span>

  • body element
  • Used for grouping and applying styles to inline elements

id

  • id is an attribute
  • Specifies a unique id for an html element
  • Can be used on any HTML element
  • When using an ID, the selector should start with '#'
