Collection Types

There are two Collection Types which are Arrays and Dictionaries


Arrays

Arrays are lists of values that are put into a variable or a constant

While defining an Array make sure to seperate values with a comma






Contains

Use the contains function to see if an array has the value you're searching for


Array Types

3 ways to state the type of an array




Repeating

You can repeat a set of string or numbers


Accessing a Specific Item

To find what value holds a position in an array assign the value to a constant then print it out


Setting a Specific Item

To set a specfic item to a specfic number on the list, set the number on the list to the value that you want to replace with

Appending

To add one item to the array, use the append function. To add multiple at the same time, use += with the items stated in []


Inserting

To add an item in a precise location, use the insert function


Removing

You can remove items from an array. If you want to remove a specfic item then use the remove at function. If you want to remove the last item then use the removeLast function. If you want to remove all items, then use the removeAll function






Adding Arrays

You can add arrays by using the + symbol




Arrays Within Arrays

you can have an array inside an array by creating an array with the selected arrays for it's values

Modifying a Dictionary
