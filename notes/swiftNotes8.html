About Classes

Classes are similar to structures as in creating a new type, but there are differences. Classes do not come with a memberwise initializer, so you have to initialize values yourself. Classes can also inherit values which is when a base class values carrie over to a sub class. Values are also directly assigned to a property instead of it being copied like in structures.

Inheritance

Inheritance is when a subclass carries over the data from a baseclass or another subclass. Variables and methods are examples of data that can be passed over.

Base Class

Base Class is the generalized class where the more specific sub classes inherit from
