HTML 3

  • <img> - defines image in a HTML page

    • <src> - source is an image attribute and it specifies the URL of an image

    There are two different ways to use this attribute

    Method 1

    Using the URL of the image

    Method 2

    Saving the image file

    1. First, the image should be saved in the folder containing the HTML code
    2. It should be a .jpg file

    The advantage of using the second method is if the image is deleted in the internet, then the URL won't work

Height & Width

Used to resize the image in pixels. A bonus is that you can specify only one height or width and the aspect ratio will be adjusted
